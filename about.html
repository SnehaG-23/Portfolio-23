"A bit about me"

Hi, I’m Sneha, a curious problem-solver and full-stack web developer who enjoys building ideas into digital experiences. I believe good code is not just about functionality — it’s about creating smooth, meaningful interactions that feel effortless to the user.

My journey into tech started with a simple curiosity about how websites actually work — what happens behind the scenes when you click a button or submit a form. That curiosity grew into a passion for mastering both the frontend magic that users see and the backend logic that powers it all.

Today, I design and develop projects that bring together:

  • Frontend skills → HTML, CSS, JavaScript, React
  • Backend skills → Python, Django, Node.js
  • Databases → MySQL
  • Other tools → REST APIs, Git

But beyond tools and frameworks, what excites me most is solving real problems — whether it’s crafting a clean, intuitive interface, optimizing performance, or structuring databases for scalability. When I’m not coding, you’ll usually find me exploring ideas for new projects, diving into tech communities, or learning the “next thing” that challenges me to grow. I see web development as more than a career — it’s a space where creativity and logic meet, and I’m driven to keep building, learning, and pushing myself forward in this ever-changing field.
